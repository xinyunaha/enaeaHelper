## 如何使用
* 下载浏览器油猴插件 Tampermonkey
* 添加新脚本
* 打开油猴插件管理面板
* 添加仓库内enaeaHelper.js的代码即可

## 特点
* 视频默认4倍加速播放，预计节省75%的时间
* 自动连播当前列表
* 自动跳转下个列表
* 真全自动学习（理论上）

## 已知问题
因为视频加速特点，会导致视频播完时，进度小于100%的情况，不用担心，列表播放一遍后插件会再次检测，对于进度不满的视频再次播放直到100%。
